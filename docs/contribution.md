# Contribution Guide

## 1. Branching
- `main` → production
- `dev` → development
- Feature branches:
  - `feature/auth`
  - `feature/posts`

## 2. Pull Requests
- Small PRs only
- Always attach screenshots (if UI)
- Add clear title + description

## 3. Commit Style
Format:
<type>: <message>

Examples:
- feat: add create post route
- fix: correct login auth
- refactor: clean post controllers
